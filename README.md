# Analisis-de-datos

**Resumen**: Repositorio con análisis y modelo para datos Saber 11. Contiene notebooks, scripts y un dashboard Streamlit para explorar predicciones.

**Cómo ejecutar el dashboard (local)**:

1. Instala dependencias:

```
pip install -r proyecto_final_saber11_para_github/requirements.txt
```

2. Inicia el dashboard:

```
streamlit run proyecto_final_saber11_para_github/app.py
```

3. Abre el navegador en la URL que muestra Streamlit (por defecto http://localhost:8501).
